# infoviz

## Project setup

Install node.js version 16.13.0. Clone project & install dependencies with the following command.

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files

npm run lint
